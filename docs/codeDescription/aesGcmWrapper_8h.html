Decrypt data using AES GSM.

AES is run as a block cypher in the GCM mode of operation. The key size is 128 bits. This function will decrypt any data length.

Parameters:
key128 bit secret key
IV96 bit initialization vector
headerAdditional authenticated data (AAD). This data is authenticated, but not decrypted.
ciphertextEncrypted data.
Returns:
plaintext Decrypted data. It is the same length as the ciphertext.
tag 128 bit authentication tag.
rtn Returns 0 if successful or else an error code
