MPin SDK and language wrappers.

This Software Development Kit (SDK) is written in C and provides implementations of these algorithms;

  1. SAKKE : The Sakai-Kasahara Key Encryption (SAKKE) algorithm uses Identity-Based Encryption to exchange a shared secret from a Sender to a Receiver. Please refer to the Internet Engineering Task Force (IETF) for further information. (rfc6508.pdf)
  2. ECCSI : The Elliptic Curve-based Certificateless Signatures for Identity-based Encryption (ECCSI) signature scheme has been implemented. Please refer to the Internet Engineering Task Force (IETF) for further information. (rfc6507)
  3. AES GCM : AES is run in Galois/Counter Mode (GCM) to provide authenticated encryption. Further details can be found in this paper


There are language wrappers provided for these popular development languages.

  1. JAVA
  2. PHP
  3. Perl

In addition example programs are provided in these languages that allow a user to verify a token that is generated by the MPin server.
