./Doxyfile            This file describes the settings to be used by the 
                      documentation system

./MPin.dox            Main page of the documentation.

To generate the documentation type: doxygen.

This will generate the directories

./html    This can be read by typing:
          firefox ./html/index.html

./latex   These are the latex source files. The pdf can be generated by
          typing: make. The output is refman.pdf.

./rtf     This contains the rich text format output.
